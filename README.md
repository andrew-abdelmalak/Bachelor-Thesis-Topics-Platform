# Bachelor Thesis Topics Platform

A React-based Single Page Application (SPA) designed to help students efficiently browse, search, and compare available Bachelor Thesis topics. 

## Features

- **Interactive Browsing**: Modern, responsive UI built with React and Tailwind CSS.
- **Search & Filtering**: Quickly find relevant thesis topics by keywords, supervisors, or categories.
- **Compare View**: Seamlessly select and compare multiple thesis topics side-by-side to make informed decisions.
- **Data-Driven**: Structured data rendering managed seamlessly via React state.

## Project Structure

```
src/
├── App.js                   # Main application entry point
├── Data.js                  # Static data store of available thesis topics
├── ThesisComparison.js      # Core component for side-by-side topic comparison
└── index.js                 # React DOM rendering
```

## Getting Started

### Prerequisites

Ensure you have Node.js and npm installed.

### Installation & Running

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Bachelor-Thesis-Topics-Platform.git
   ```
2. Navigate into the directory:
   ```bash
   cd Bachelor-Thesis-Topics-Platform
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

The application will be available at `http://localhost:3000`.
